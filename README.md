# 🤖 Série Robótica Criativa – Volume 1

### Fundamentos e Projetos Práticos com Arduino

Bem-vindo ao repositório oficial de códigos do **Curso de Robótica Educacional** da **ACNP** (Associação Cultural Nascente Pequena). Aqui você encontra todos os *sketches* (códigos) utilizados nos 15 projetos práticos da nossa apostila.

## 📚 Sobre o Material

Este repositório serve de apoio para o Volume 1 da nossa série didática. O objetivo é facilitar o aprendizado, permitindo que alunos e professores baixem, testem e modifiquem os códigos apresentados em sala de aula.

## 🛠️ Como Usar

1. **Baixe o Código:** Clique no botão verde "Code" acima e escolha "Download ZIP", ou clone o repositório se souber usar Git.

2. **Abra na IDE:** Abra o arquivo `.ino` do projeto desejado na IDE do Arduino.

3. **Conecte e Carregue:** Conecte seu Arduino Uno ao computador via USB, selecione a porta correta e clique em "Carregar" (Seta para direita).
